h1. Mohiva Common "!http://travis-ci.org/mohiva/common.png!":http://travis-ci.org/mohiva/common

The Mohiva Common project is a PHP library which provides functionality shared by all mohiva projects.

h2. Installation

You can install the library through "composer":http://getcomposer.org. All you need is to add it as dependency to your composer.json.

<pre lang="js">
{
    "require": {
        "mohiva/common": "dev-master"
    }
}
</pre>

h2. Requirements

Mohiva Common needs PHP 5.4 to work.
